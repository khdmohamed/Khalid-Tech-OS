# Skill: Shopify GraphQL

Generate Shopify GraphQL queries and mutations for the Admin or Storefront API.

## Context
- Always specify API version (2024-10 or latest stable)
- Note rate limit implications for bulk operations
- Flag webhook requirements where relevant
- For Arabic RTL: flag metafield and theme liquid implications

## Common Operations
- Product sync (create/update/delete)
- Order fetch and fulfillment
- Customer account data
- Inventory level updates
- Price rules and discounts
